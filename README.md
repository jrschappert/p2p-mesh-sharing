# P2P Generative 3D Mesh Sharing

This project uses a P2P network to share 3D meshes between peers in an effort to offer a more scalable and cost-effective way to share user-generated assets among many users.

View the deployed project here: https://jrschappert.github.io/p2p-mesh-sharing/

Refer to DeveloperSetupGuide.md for instructions on how to run this locally.
